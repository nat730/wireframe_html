<a>

Utilisée pour les hyperliens. Attention : l'attribut "name" n'existe plus pour les liens.

Attributs
nom description valeurs(exemple)
href Cet attribut spécifie l'URL de destination vers laquelle le lien doit pointer. <a href="example">Lien vers Example</a>.
target Cet attribut spécifie comment le lien doit être ouvert lorsqu'il est cliqué. Les valeurs courantes pour cet attribut sont _blank (ouvre le lien dans une nouvelle fenêtre ou un nouvel onglet), _self (ouvre le lien dans la même fenêtre ou le même onglet), _parent (ouvre le lien dans le cadre parent, utilisé dans les cadres) et _top (ouvre le lien dans la fenêtre complète, annulant les cadres). <a target="_blank|_self|_parent|_top|framename">
title Cet attribut fournit un texte supplémentaire (info-bulle) qui apparaît lorsque vous survolez le lien avec la souris. <a title="World Health Organization">WHO</a>

exemple

<a href="https://example.com">Website </a>

<abbr>

Pour une abréviation. Son attribut "title" ne doit être utilisé que pour l'extension de l'abréviation et rien d'autre.

Attributs
nom description valeurs(exemple)
lang Cet attribut spécifie la langue de l'abréviation ou de l'acronyme. Il est utilisé pour les sites multilingues où différentes versions de l'abréviation peuvent être fournies pour chaque langue. <abbr title="World Wide Web" lang="en">WWW </abbr>
classs/id Ces attributs permettent de définir des classes ou des identifiants CSS pour la balise <abbr>, ce qui permet de styliser l'abréviation avec du CSS ou de la cibler avec du JavaScript. <abbr id="abbr">tests<abbr>

Généralement utilisée dans le conteneur <footer>, cette balise est utilisée à la place de la balise <p>, les informations nécessaires pour contacter la ou les auteurs du document cité. Confusion possible : on utilisera la balise <p> pour donner l'adresse postale d'une entreprise sans relation avec l'auteur du document s'y rapportant.
